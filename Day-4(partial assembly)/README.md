# Gearbox Assembly Design

## Overview

This repository contains the CAD design and assembly of a gearbox created in SolidWorks. The project consists of multiple gears, shafts, and a housing assembled to demonstrate the fundamentals of mechanical power transmission.

The design focuses on gear meshing, shaft alignment, and assembly techniques commonly used in gearbox systems. It serves as a practical exercise for improving CAD modeling and mechanical design skills.

## Components Used

* Gearbox Housing
* Spur Gears
* Input Shaft
* Intermediate Shafts
* Output Shaft

## Features

* Complete gearbox assembly
* Multiple meshing gears
* Detailed housing design
* Proper shaft and gear alignment
* Parametric SolidWorks model
* Motion study ready

## Working Principle

The gearbox transfers rotational motion from the input shaft through a series of meshing gears to the output shaft. Depending on the gear arrangement and gear ratios, the system can modify speed and torque to meet application requirements.

This assembly demonstrates the basic principles of gear trains and mechanical power transmission.
